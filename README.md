# pyx2pxd


Simple library to automatically to auto generate pxd files from pyx by extracting function and class signatures.

---


## Installation

You can install `pyx2pxd` via [pip]:

```shell
pip install pyx2pxd
```

## Running

```shell
pyx2pxd path_with_pyx_files
```
